プライバシーポリシー / Privacy Policy

最終更新日 / Last Updated: 2025年1月24日 / January 24, 2025

1. 情報収集 / Information Collection

AI Playgroundでは、以下の情報を収集する場合があります：

  • アップロードされたCSVファイル（一時的に保存）
  • ブラウザの基本情報（ユーザーエージェント、IPアドレス）
  • セッション情報（アプリケーションの使用状況）

AI Playground may collect the following information:

  • Uploaded CSV files (temporarily stored)
  • Basic browser information (user agent, IP address)
  • Session information (application usage)

2. 情報の使用 / Use of Information

収集した情報は以下の目的でのみ使用されます：

  • 機械学習モデルの学習と予測
  • アプリケーションの改善
  • 技術的な問題の解決

Collected information is used only for the following purposes:

  • Machine learning model training and prediction
  • Application improvement
  • Technical issue resolution

3. 情報の共有 / Information Sharing

お客様の個人情報を第三者に販売、貸与、または提供することはありません。 ただし、以下の場合を除きます：

  • 法的義務がある場合
  • お客様の明示的な同意がある場合
  • サービス提供に必要な場合（データ処理業者など）

We do not sell, rent, or provide your personal information to third parties, except in the following cases:

  • When required by law
  • When you give explicit consent
  • When necessary for service provision (data processors, etc.)

4. データの保存期間 / Data Retention

アップロードされたファイルは、セッション終了後24時間以内に自動的に削除されます。 セッション情報は、ブラウザを閉じるか、手動でリセットした時点で削除されます。

Uploaded files are automatically deleted within 24 hours after the session ends. Session information is deleted when you close your browser or manually reset it.

5. セキュリティ / Security

お客様のデータを保護するため、以下のセキュリティ対策を実施しています：

  • HTTPS暗号化通信
  • セキュアなファイル処理
  • 定期的なセキュリティ監査

To protect your data, we implement the following security measures:

  • HTTPS encrypted communication
  • Secure file processing
  • Regular security audits

6. お客様の権利 / Your Rights

お客様には以下の権利があります：

  • データの削除を要求する権利
  • データの使用停止を要求する権利
  • プライバシーポリシーの変更について通知を受ける権利

You have the following rights:

  • Right to request data deletion
  • Right to request cessation of data use
  • Right to be notified of privacy policy changes

7. お問い合わせ / Contact

プライバシーポリシーに関するご質問やご要望がございましたら、 以下の方法でお問い合わせください：

Email: privacy@ai-playground.com
GitHub: Issue Tracker

If you have any questions or requests regarding this privacy policy, please contact us through:

Email: privacy@ai-playground.com
GitHub: Issue Tracker
