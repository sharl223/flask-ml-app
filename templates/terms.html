AI Playgroundは、初心者向けの機械学習Webアプリケーションです。 以下の機能を提供します：

  • 線形回帰モデルの作成・学習・予測

  • LightGBM機械学習モデルの作成・学習・予測

AI Playground is a beginner-friendly machine learning web application that provides the following features:

  • Linear regression model creation, training, and prediction
